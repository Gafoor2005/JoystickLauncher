## ğŸš€ Quick Start

### Installation

1. Download the latest release from [Releases](../../releases)
2. Enable "Install from unknown sources" in Settings
3. Install the APK
4. Set as default launcher
5. Grant notification permissions

For detailed installation instructions, see [INSTALLATION.md](INSTALLATION.md).

### Usage

- **Open App Drawer**: Swipe up from home screen
- **Use Joystick**: Drag the joystick to select and launch apps
- **View Notifications**: Swipe down from top
- **Navigate**: Use gestures and taps

For complete usage guide, see [USER_GUIDE.md](USER_GUIDE.md).
